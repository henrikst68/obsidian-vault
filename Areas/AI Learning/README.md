## How this works

Three kinds of notes, each with a template:

- **Concept notes** — atomic, one idea per note. Linkable from everywhere. → [[Templates/Concept Template]]
- **Resource notes** — one per article, paper, video, or talk you consume. → [[Templates/Resource Template]]
- **Experiment notes** — one per thing you build or try. Hypothesis, what happened, what you learned. → [[Templates/Experiment Template]]

The loop: read something → make a Resource note → extract any new ideas into Concept notes → when you've got enough of a hunch, start an Experiment.
